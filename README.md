Library Search Engine
---
> Lancer le back-end
#### Prérequis Sping Boot, Java 21
```bash
cd Library-search-engine/backend/src
mvn spring-boot:run
```

> Lancer le front-end
#### Prérequis Angular
```bash
cd Library-search-engine/frontend
npm install
ng serve
```
#### Aller sur http://localhost:4200/
